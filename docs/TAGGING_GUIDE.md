# 標籤使用指南

> 本指南幫助您為網站內容選擇合適的標籤，保持標籤系統的一致性和可維護性。

## 標籤層級系統

我們使用三層標籤系統，讓內容分類既清晰又靈活：

### 一級標籤：內容類型（必選一個）

這些標籤定義了文章的主要目的：

- `論文閱讀` - 學術論文的閱讀筆記（含「我的理解」）
- `技術分享` - AI、技術工具、方法論的分享
- `研究發現` - 研究結果、數據分析的分享
- `教學資源` - 教程、指南、入門內容

**範例**：
```yaml
tags:
  - 論文閱讀  # 一級標籤
```

### 二級標籤：主題領域（可選多個）

這些標籤描述文章涉及的學科或領域：

**社會科學**：
- `心理學`
- `經濟學`
- `政治學`
- `社會學`

**科技領域**：
- `人工智慧`
- `數據科學`
- `政治科技`
- `數位治理`

**方法論**：
- `行為遺傳學`
- `演化心理學`
- `資訊經濟學`

**範例**：
```yaml
tags:
  - 論文閱讀
  - 心理學      # 二級標籤
  - 演化心理學  # 二級標籤
```

### 三級標籤：具體技術/概念（可選多個）

這些標籤標記具體的技術、模型、或研究方法：

**AI 相關**：
- `大語言模型` 或 `LLM`
- `強化學習`
- `RLVR`
- `提示工程`
- `ChatGPT`
- `DeepSeek`

**研究方法**：
- `GWAS`（全基因組關聯研究）
- `PNAS`（期刊名）
- `arXiv`

**其他概念**：
- `參與式預算`
- `社會流動`
- `監控資本主義`
- `GPU 計算`

**範例**：
```yaml
tags:
  - 論文閱讀
  - 行為遺傳學
  - GWAS          # 三級標籤
  - 演化心理學
```

## 命名規範

### 語言選擇

1. **優先使用繁體中文**
   - ✅ `人工智慧`、`心理學`、`論文閱讀`
   - ❌ `AI`（除非是常見縮寫）、`psychology`

2. **常見縮寫使用英文大寫**
   - ✅ `LLM`、`AI`、`GWAS`、`RLVR`
   - ❌ `llm`、`ai`（小寫）

3. **專有名詞保持原文**
   - ✅ `ChatGPT`、`DeepSeek`、`PNAS`、`arXiv`
   - ❌ `聊天GPT`

### 一致性原則

同一概念只使用一個標籤：
- ✅ 統一使用 `人工智慧` 或 `AI`（建議前者）
- ❌ 同時使用 `人工智慧` 和 `AI`

## 標籤數量建議

- **最少**：2 個（一級 + 二級）
- **推薦**：3-6 個
- **最多**：8 個（避免過度標記）

## 常見問題

### Q: 應該用「AI」還是「人工智慧」？
A: 推薦使用 `人工智慧`（與網站主語言一致）。但 `AI` 作為常見縮寫也可接受。選定後保持一致。

### Q: 論文閱讀和研究發現有什麼區別？
A: 
- `論文閱讀`：閱讀他人論文後的筆記（包含「我的理解」）
- `研究發現`：分享研究結果或數據分析

### Q: 英文標籤需要大小寫嗎？
A: 
- 縮寫使用大寫：`LLM`、`AI`、`GWAS`
- 專有名詞保持原樣：`ChatGPT`、`DeepSeek`

### Q: 可以創建新標籤嗎？
A: 可以！但建議先檢查是否有相似的現有標籤，避免重複。
